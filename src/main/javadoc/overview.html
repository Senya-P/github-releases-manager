Github Releases Manager

The GitHub Releases Manager is a command-line tool that allows users to search for, install, uninstall and update GitHub releases across different platforms.

Overview

This tool provides an abstraction over the GitHub API and platform-specific installation logic to simplify retrieving GitHub releases.

Terminology

  • Release: A versioned snapshot of a software in a GitHub repository, typically containing compiled binaries.
  • Asset: A downloadable file associated with a release, such as a binary or installer.
  • Repository: A GitHub repository that may contain published releases.
  • Platform: The operating system where the asset will be installed.

Core Functionality

  • Search: Find public repositories by name.
  • Install: Download and install the suitable asset of the latest release from a given repository.
  • Uninstall: Remove an installed asset.
  • Update: Update an installed asset to the latest version.
  • List: Show all currently installed assets.
  • Help: Display usage instructions.

Architecture

The entry point is the Main class, which creates an instace of ReleaseManager, which coordinates command parsing via CmdParser, GitHub API interaction via GithubClient, and platform-specific behaviors via the PlatformHandler.
